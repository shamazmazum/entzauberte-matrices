Overview

Entzauberte (something like "disenchanted") matrices is a lightweight wrapper around LAPACK. It can add/multiply/invert matrices, has a solver for systems of linear equations and can perform basic matrix decompositions.

In contrast to MAGICL, entzauberte-matrices operates on usual Common Lisp arrays with real or complex floating points elements, more precisely on arrays of type (simple-array TYPE 2) where TYPE is one of single-float/double-float/(complex single-float)/(complex double-float). For binary operations, both arguments must have the same type.

Currently, entzauberte-matrices is SBCL only library. It requires OpenBLAS to work.
