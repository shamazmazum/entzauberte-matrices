API

dot(v1 v2)
Compute the dot product \(\langle v_1, v_2 \rangle = \sum_i {v_1}_i \overline{{v_2}_i}\).
norm(v)
Compute the norm of \(v\), i.e. \(\sqrt{\langle v, v\rangle}\).
add(m1 m2 &optional (s (coerce 1 (array-element-type m1))))
Compute \(s m_1 + m_2\) where \(s\) is a scalar and \(m_1\) and \(m_2\) are two matrices of vectors.. By default \(s\) is \(1\).
sub(m1 m2)
Compute \(m_1 - m_2\). A special case of add.
scale(m s)
Compute \(s m\) where \(s\) is a scalar and \(m\) is a matrix or a vector.
mult(a b &key ta tb (scale (coerce 1 (array-element-type a))))
Compute \(s \hat{A}\hat{B}\) where \(\hat{A}\) (resp. \(\hat{B}\)) is \(A\)(resp. \(B\)) if :ta (resp. :tb) is nil and \(A^T\)(resp. \(B^T\)) otherwise and \(s\) is a scalar.
@(&rest ms)
Left associative multiplication of matrices.
det(m)
Compute \(\det (M)\) using \(LU\) factorization. Returns (values det infocode) where det can be NIL if \(LU\) factorization fails.
invert(m)
Compute \(M^{-1}\) using \(LU\) factorization. Returns (values inv infocode) where inv can be NIL if \(LU\) factoirization or inversion routine fails.
eig(m)

Compute eigenvalues and eigenvectors of \(M\). Eigenvectors are stored in rows, so that (in pseudo-code)

(multiple-value-bind (vals vecs)
    (eig m)
  (approx= (mult vecs (to-complex m))
           (mult (to-complex (from-diag vals)) vecs)))

is T. The returned values vals and vecs may be NIL if the decomposition fails. The third returned value is the info code returned by LAPACK.

eig-self-adjoint(m where)

Compute eigenvalues and eigenvectors of a self-adjoin matrix \(M\). Eigenvectors are stored in rows, so that (in pseudo-code)

(multiple-value-bind (vals vecs)
    (eig m)
  (approx= (mult vecs m)
           (mult (from-diag vals) vecs)))

is T. The returned values vals and vecs may be NIL if the decomposition fails. The third returned value is the info code returned by LAPACK.

svd(m &key compact)

Do the singular value decomposition of a matrix \(M = U\Sigma V^T\). Matrices \(U\), \(\Sigma\) and \(V^T\) are returned in three values. For a matrix \(m\times n\) and \(r = \min(m, n)\), if :compact is NIL then \(U\) is an orthogonal \(m \times m\) matrix, \(V^T\) is an orthogonal \(n \times n\) matrix and \(\Sigma\) is a vector of the length \(r\), so that in pseudocode

(multiple-value-bind (U Σ VT)
    (svd m)
  (approx= (mult U (mult (from-diag Σ m n) VT)) m))

is T. If :compact is T then \(U\) is a semi-orthogonal \(m \times r\) matrix, \(V^T\) is a semi-orthogonal \(r \times n\) matrix and \(\Sigma\) is as with :compact being NIL, so that in pseudocode

(multiple-value-bind (U Σ VT)
    (svd m)
  (let ((r (min (array-dimension m 0)
                (array-dimension m 1))))
    (approx= (mult U (mult (from-diag Σ r r) VT)) m)))

is T. The first three returned values may be NIL if the decomposition fails. The fourth value is the info code returned by LAPACK.

transpose(m)
Transpose a matrix \(M\)(slow for big matrices, avoid this function if possible).
solve(a b)
Solve an equation \(AX = B\). This function uses transpose internally. Return (values X infocode) where X may be NIL if the solver fails.
reshape(m shape)
Reshape an array. Deprecated: use column, row, vector->column or vector->row.
reshape-unsafe(m shape)
Reshape an array. The new array will share the storage with the argument. Deprecated: use column, row, vector->column-unsafe or vector->row-unsafe.
vector->column(m)
Turn a vector to a \(n \times 1\) matrix.
vector->column-unsafe(m)
Turn a vector to a \(n \times 1\) matrix. The result will share the storage with m.
vector->row(m)
Turn a vector to a \(1 \times n\) matrix.
vector->row-unsafe(m)
Turn a vector to a \(1 \times n\) matrix. The result will share the storage with m.
hstack(ms type)
Concatenate matrices horizontally. The matrices must have the same number of rows and the same element type type.
vstack(ms type)
Concatenate matrices vertically. The matrices must have the same number of columns and the same element type type.
map(f element-type m)
Apply an unary operation f to a matrix m element-wise. element-type must match the element type of the result.
zip-with(f element-type x y)
Apply a binary operation f to matrices x and y element-wise. element-type must match the element type of the result.
set-num-threads(n)
Set a number of threads for OpenBLAS.
